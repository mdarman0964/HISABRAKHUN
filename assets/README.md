# Assets Folder

This folder contains all the static assets for the Hisab app.

## Required Files

### App Icons
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (512x512)

### Splash Screen
- `splash.png` - Splash screen image (1242x2436)

### Images
- Add your app images here

### Fonts
- Add custom fonts here

## Icon Generation

Use the following tools to generate app icons:

1. **App Icon Generator**: https://appicon.co/
2. **Android Asset Studio**: https://romannurik.github.io/AndroidAssetStudio/

## Splash Screen Guidelines

- Size: 1242x2436 (iPhone X size)
- Background: Solid color matching app theme (#1a1a2e)
- Center logo: 300x300 or smaller
- Keep it simple and minimal
